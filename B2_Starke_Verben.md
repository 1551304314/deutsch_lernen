# B2 强变化动词大全 (Starke Verben)

> 按 Ablaut 元音变化规律分组，含现在时特殊变形（du / er-sie-es）、过去时、完成时

---

## 缩写说明
- **Präs.** = 现在时特殊变形 (du 与 er/sie/es 的元音变化)
- **Prät.** = Präteritum 过去时 (第三人称单数)
- **Perf.** = Partizip II 完成时分词
