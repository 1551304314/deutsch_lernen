## 一、形容词变格第三类：alle / beide / sämtliche / viele / wenige / einige / mehrere

### 核心规则

| 前导词 | 复数 N/A 形容词词尾 | 复数 Dat 形容词词尾 | 备注 |
|------|:--:|:--:|------|
| alle | -en | -en | 严格弱变化 |
| beide | -en | -en | 同 alle |
| sämtliche | -en | -en | 同 alle |
| viele | -e 或 -en | -en | 书面偏 -en，口语偏平行 |
| wenige | -e 或 -en | -en | 同 viele |
| einige | -e 或 -en | -en | 同 viele |
| mehrere | -e 或 -en | -en | 同 viele |

### viel / wenig + 不可数名词（单数）— 强变化

| | viel + 阳 | viel + 中 | viel + 阴 |
|:--:|------|------|------|
| Nom | viel guter Wein | viel gutes Wasser | viel gute Milch |
| Akk | viel guten Wein | viel gutes Wasser | viel gute Milch |
| Dat | viel gutem Wein | viel gutem Wasser | viel guter Milch |

### 记忆口诀
- 前导词带词尾 -e → 形容词 -en（弱变化）
- 前导词无词尾 → 形容词强变化（带格标记）

## 二、Komparativ und Superlativ 比较等级全体系

### 基本公式

| 等级 | 构成 | 例（klein/schnell） |
|:--:|------|------|
| 原级 Positiv | 原形 | klein |
| 比较级 Komparativ | 原形 + -er | klei**ner** |
| 最高级 Superlativ | am + -sten | am kleins**ten** |

### 最高级两种形式

| 形式 | 用途 | 例 |
|------|------|------|
| am + -sten | 表语/副词 | am größten |
| der/die/das + -ste | 定语（后接名词） | der größte Mann |

### 三大变音规则（单音节词）

| 变音 | Positiv → Komparativ | 例 |
|:--:|------|------|
| a → ä | alt → älter | warm → wärmer |
| o → ö | groß → größer | hoch → höher |
| u → ü | jung → jünger | kurz → kürzer |

### 特例拼写

| 情况 | 规则 | 例 |
|------|------|------|
| -d/-t/-s/-ß/-z/-sch 结尾 | Superlativ 加 e | heiß → am heiß**e**sten |
| -el/-er 结尾 | Komparativ 去 e | dunkel → dunk**ler** |

### 不规则必背

| Positiv | Komparativ | Superlativ | 中文 |
|------|------|------|------|
| gut | besser | am besten | 好 |
| gern | lieber | am liebsten | 喜欢 |
| viel | mehr | am meisten | 多 |
| hoch | höher | am höchsten | 高 |
| nah | näher | am nächsten | 近 |

## 三、同级比较：so / genauso / ebenso ... wie

| 表达 | 强度 | 例 |
|:--:|:--:|------|
| so ... wie | 中性 | so groß wie |
| genauso ... wie | 强调 = 完全一样 | genauso groß wie |
| ebenso ... wie | 正式书面 | ebenso groß wie |

否定：**nicht so ... wie**（不如……）

## 四、je ... desto / umso（越……越……）

```
je + Komparativ（从句，动词末尾）, desto/umso + Komparativ（主句，动词二位）
```

| 例 | 中文 |
|------|------|
| Je mehr ich lerne, desto mehr weiß ich. | 学得越多，就知道得越多。 |
| Je älter er wird, desto ruhiger wird er. | 年纪越大，越冷静。 |

简化版：**immer + Komparativ**（同一主体渐进 → 「越来越……」）
- Es wird **immer wärmer**. → 越来越暖。

## 八、Konsekutivsätze 结果从句（A2→B1）⭐ 今日重点练习

### 五大结构

| 结构 | 用法 | 例 | 级别 |
|------|------|------|:--:|
| **sodass** | 所以/以至于 | ... sodass das Spiel abgesagt wurde. | B1 |
| **so + Adj + dass** | 如此……以至于 | so langweilig, dass ich einschlief | B1 |
| **zu + Adj + um...zu** | 太……而不能（主语一致） | zu kalt, um zu schwimmen | A2 |
| **zu + Adj + als dass** | 太……以至于不能（主语可不同）+ K II | zu kalt, als dass man schwimmen könnte | B1/B2 |
| **genug + um...zu** | 足够……可以 | alt genug, um zu verstehen | A2/B1 |

## 九、Konzessivsätze 让步从句（A2→B2）⭐ 全新

### 完整递进表

| 连词/结构 | 含义 | 语气 | 级别 | 动词位置 |
|------|------|:--:|:--:|:--:|
| **obwohl** | 虽然（事实） | 弱 | A2 | 末尾 |
| **auch wenn** | 即使（假设） | 中 | B1 | 末尾 |
| **selbst wenn** | 就算/哪怕 | 强 | B1/B2 | 末尾（常 K II） |
| **wenn auch** | 虽然/即使 | 中 | B1/B2 | 末尾 |
| **so ... auch** | 无论多么 | 强 | B2 | 末尾或倒装 |
| **wer/was/wie auch** | 无论谁/什么/怎样 | 强 | B2 | 两可 |
| obgleich/obschon/wenngleich | 虽然（书面） | — | B2 | 末尾 |

### obwohl vs trotzdem 辨析

| | obwohl | trotzdem |
|:--:|------|------|
| 词性 | 从句连词 | 副词 |
| 动词 | 末尾 | 第二位 |
| 例 | Obwohl es regnet, gehe ich. | Es regnet. Trotzdem gehe ich. |

> ❌ 不能同时用：~~Obwohl es regnet, trotzdem gehe ich.~~

### so ... auch 两种语序

| 版本 | 公式 | 例 |
|:--:|------|------|
| A（从句式）| So + Adj + 主语 + ... + auch + 动词**末尾** | So schwer es auch **ist**, ... |
| B（倒装式）| So + Adj + 动词**二位** + 主语 + ... + auch | So schwer **ist** es auch, ... |

### trotz + Genitiv 介词版

```trotz + Genitiv = obwohl 从句的名词化```
- **Trotz des Regens** gehe ich spazieren. = **Obwohl es regnet**, gehe ich spazieren.
