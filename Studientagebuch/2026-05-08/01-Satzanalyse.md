# Satzanalyse: "Nur wer keine Zukunft hat, redet über seine Vergangenheit."

> 只有没有未来的人，才谈论自己的过去。

## 单词

| 单词 | 词性 | 含义 |
|------|------|------|
| nur | 副词 | 仅仅、只有 |
| wer | 疑问/关系代词 | 谁 / ……的人 |
| keine | 否定冠词（宾格阴性） | 没有 |
| Zukunft | 名词（阴） | 未来 |
| hat | 动词（haben 第三人称单数） | 有 |
| redet | 动词（reden 第三人称单数） | 说话、谈论 |
| über | 介词（+宾格） | 关于 |
| seine | 物主代词（宾格阴性/复数） | 他的 / 自己的 |
| Vergangenheit | 名词（阴） | 过去 |

---

## 语法要点

### 1. "wer..." 作为泛指关系从句（主语从句）

`wer` 不指某个特定的人，而是泛指「……的人」，相当于 "derjenige, der..."

整个从句 `wer keine Zukunft hat` 充当主句的**主语**。

### 2. 否定词 kein-

否定不带定冠词的名词时用 `kein-`，而非 `nicht`。

`die Zukunft → keine Zukunft`（宾格阴性，所以用 **keine**）

### 3. reden über + 宾格

固定搭配，"谈论某事"，`über` 后接第四格（宾格）。

### 4. 物主代词 seine

指代从句首的 `wer`（泛指的"那个人"），意为"他自己的"。

`die Vergangenheit` 是阴性宾格，所以用 `seine`（不是 seinen 或 sein）。
