# 对话练习 — Einkaufen 购物 - 2026-05-11

## 学习时间
2026-05-11

## 内容总结
A2 核心场景：购物对话 (Einkaufen)，覆盖面包店 (Bäckerei) 和超市 (Supermarkt) 两个场景。

---

## 场景一：In der Bäckerei（在面包店）

### 完整对话

| 角色 | 德语 | 中文 |
|------|------|------|
| 👩‍🦰 Kunde | Guten Tag! | 您好！ |
| 🧑‍🍳 Verkäufer | Guten Tag! Was darf es sein? | 您好！您想要什么？ |
| 👩‍🦰 Kunde | Ich möchte vier Brötchen, bitte. | 我想要四个小面包。 |
| 🧑‍🍳 Verkäufer | Sonst noch etwas? | 还要别的吗？ |
| 👩‍🦰 Kunde | Ja, noch zwei Croissants, bitte. | 是的，还要两个牛角面包。 |
| 🧑‍🍳 Verkäufer | Das macht 4,80 Euro. | 一共4.80欧元。 |
| 👩‍🦰 Kunde | Kann ich mit Karte bezahlen? | 我可以用卡支付吗？ |
| 🧑‍🍳 Verkäufer | Ja, natürlich. | 当然可以。 |
| 👩‍🦰 Kunde | Hier, bitte. | 给您。 |
| 🧑‍🍳 Verkäufer | Danke, auf Wiedersehen! | 谢谢，再见！ |

### 重点表达

| 表达 | 说明 |
|------|------|
| Was darf es sein? | 店员常用语 = 您想要什么？ |
| Sonst noch etwas? | 还要别的吗？ |
| Das macht ... Euro. | 一共……欧元 |
| Kann ich mit Karte bezahlen? | 我可以用卡支付吗？ |

## 场景二：Im Supermarkt（在超市）

### 问价对话

| 顾客 | 店员 |
|------|------|
| Entschuldigung, was kostet ein Kilo Äpfel? | 2,50 Euro. |
| Und die Bananen? | 1,80 Euro. |
| Ich finde die Äpfel zu teuer. | Es tut mir leid. |

## 场景三：超市找东西

| 顾客 | 店员 |

## 今日购物句型总结

| 德语 | 中文 |
|------|------|
| Was darf es sein? | 您想要什么？ |
| Sonst noch etwas? | 还要别的吗？ |
| Das macht ... Euro. | 一共……欧元。 |
| Was kostet ...? | ……多少钱？ |
| Kann ich mit Karte bezahlen? | 我可以用卡支付吗？ |
| Ich suche ... | 我在找…… |
| Ich finde ... zu teuer. | 我觉得……太贵了。 |
| Ich möchte ... kaufen. | 我想要买…… |
