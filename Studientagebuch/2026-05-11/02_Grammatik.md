## 一、名词复数六大规则

### 总表

| 类型 | 复数标记 | 变音？ | 举例 | 常见于 |
|------|---------|--------|------|--------|
| ① | **-e** | 可有 Umlaut | Tisch→Tische, Stuhl→Stühle | 阳/中性名词 |
| ② | **-en / -n** | 无 | Frau→Frauen, Tasche→Taschen | 阴性名词 |
| ③ | **-er** | 必须有 | Kind→Kinder, Buch→Bücher | 中性名词 |
| ④ | **-s** | 无 | Auto→Autos, Handy→Handys | 外来词 |
| ⑤ | **无词尾** | 可有 Umlaut | Apfel→Äpfel, Brötchen→Brötchen | 阳性、-chen/-lein |
| ⑥ | **无变化** | 无 | Lehrer→Lehrer | -er 结尾阳性 |

### 记忆口诀
```
阳名大多加 -e，
阴名通通加 -en，
中名爱加 -er 玩，
外来词汇加 -s 算，
-chen -lein 永不变，
Apfel Äpfel 光头换！
```

## 二、形容词变格三步法

### 核心判断流程
```
第一步：看有没有冠词？
     有 → 第二步
     没 → 强变化（形容词复制 der/die/das 词尾）

第二步：定冠词还是不定冠词？
     定冠词 (der/die/das) → 弱变化（5个 -e，其余全 -en）
     不定冠词 (ein/eine) → 混合变化

第三步（混合变化）：ein 有词尾吗？
     有 → 弱变化 -e/-en
     没有（光头）→ 强变化 -er/-es
```

### 强变化表（无冠词）

| 格 | 阳性 | 中性 | 阴性 | 复数 |
|----|:--:|:--:|:--:|:--:|
| Nom | -er | -es | -e | -e |
| Akk | -en | -es | -e | -e |
| Dat | -em | -em | -er | -en |

> 口诀：形容词直接复制定冠词词尾

### 弱变化表（定冠词后）

| 格 | 阳性 | 中性 | 阴性 | 复数 |
|----|:--:|:--:|:--:|:--:|
| Nom | -e | -e | -e | -en |
| Akk | -en | -e | -e | -en |
| Dat | -en | -en | -en | -en |

> 口诀：五个 -e（Nom 阳中阴 + Akk 中阴），其余全 -en

### 混合变化（不定冠词后）

| 位置 | ein 有词尾？ | 形容词 |
|------|:---:|------|
| 阳 Nom: ein \_\_\_ Wein | ❌ 光头 | 补 -er（强） |
| 中 Nom: ein \_\_\_ Bier | ❌ 光头 | 补 -es（强） |
| 中 Akk: ein \_\_\_ Bier | ❌ 光头 | 补 -es（强） |
| 阴 Nom: eine \_\_\_ Milch | ✅ -e | -e（弱） |
| 阳 Akk: einen \_\_\_ Wein | ✅ -en | -en（弱） |

> 口诀：ein 光头我不光，ein 有尾我偷懒。

## 三、mit 等九大 Dativ 介词

### Dativ 介词清单

| 介词 | 中文 | 格 |
|------|------|:--:|
| **mit** | 和、用、带着 | Dat |
| **aus** | 从……出来 | Dat |
| **bei** | 在……附近、在……处 | Dat |
| **nach** | 向、在……之后 | Dat |
| **von** | 从、……的 | Dat |
| **zu** | 到、去 | Dat |
| **seit** | 自从 | Dat |
| **gegenüber** | 在……对面 | Dat |
| **außer** | 除了 | Dat |

### mit 的三个核心用法

| 用法 | 例句 |
|------|------|
| 工具/方式（用） | mit Karte bezahlen（刷卡支付） |
| 陪同（和……一起） | mit meiner Mutter einkaufen（和我妈购物） |
| 特征（带着） | mit einem roten Preis（带着红标价） |

### 口诀
> aus, bei, mit, nach, seit, von, zu — 三格永远不分离！

## 四、德语 vs 英语语法描述传统

### 书写顺序差异
| 德语 | 英语 |
|------|------|
| etwas (Akk) suchen | to look for something |
| jemandem (Dat) helfen | to help someone |

### 原因
德语是 **动词居尾语言**（Verb-final），在 zu 不定式、完成时、从句中动词天然在末尾。辞典写法反映的是德语的"内在语序"。
