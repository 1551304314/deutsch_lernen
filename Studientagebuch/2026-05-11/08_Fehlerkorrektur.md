## 一、Perfekt 复习测试错误

### 错误 1：weil 从句 + Perfekt 助动词位置

| 项目 | 内容 |
|------|------|
| 错误 | weil ich heute früh aufgestanden |
| 正确 | weil ich heute früh aufgestanden **bin** |
| 规则 | weil 从句 + Perfekt：助动词 bin/habe 放从句最末尾，排在 Partizip II 之后 |

### 错误 2：Perfekt 主句语序

| 项目 | 内容 |
|------|------|
| 错误 | Er hast getrunken ein Glas Milch. |
| 正确 | Er **hat** ein Glas Milch **getrunken**. |
| 规则 | (1) er + haben → hat (不是 hast)；(2) Perfekt 主句 Partizip II 在句尾 |

### 错误 3：gehen Partizip II 拼写

| 项目 | 内容 |
|------|------|
| 错误 | Wir sind gestern ins Kino gagangen. |
| 正确 | Wir sind gestern ins Kino **gegangen**. |

## 二、名词复数错误

### 错误 4：Apfel → Äpfel

| 项目 | 内容 |
|------|------|
| 错误 | zwei Croissant（少复数 -s）|
| 正确 | zwei Croissant**s** |
| 规则 | 外来词复数加 -s |
| 错误语境 | 面包店点餐 |

### 错误 5：Äpfel 复数

| 项目 | 内容 |
|------|------|
| 错误 | ein Kilo Apfel |
| 正确 | ein Kilo **Äpfel** |
| 规则 | 一类苹果用复数：Apfel → Äpfel（⑤ 光头变音型 a→ä） |

### 错误 6：复数 Äpfel 拼写

| 项目 | 内容 |
|------|------|
| 错误 | Findest du die Apfel zu teuer? |
| 正确 | Findest du die **Äpfel** zu teuer? |

## 三、形容词变格测试错误

### 错误 7：混合变化 ein 光头

| 项目 | 内容 |
|------|------|
| 错误 | Ein gute Preis! |
| 正确 | Ein **guter** Preis! |
| 规则 | 阳 Nom + ein 光头 → 形容词补强变化 -er |

### 错误 8：Dativ 强变化

| 项目 | 内容 |
|------|------|
| 错误 | mit alten Karte |
| 正确 | mit **alter** Karte |
| 规则 | 无冠词 Dativ 阴 → 强变化 -er |
