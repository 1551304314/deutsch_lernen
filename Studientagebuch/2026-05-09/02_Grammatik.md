## 二、可分/不可分动词

### 识别口诀

**可分词前缀**：auf, zu, ein, aus, mit, nach, vor, an, ab, los, weg — 句尾躲！

**不可分前缀**：be-, emp-, ent-, er-, ge-, miss-, ver-, zer- — 永不分离！

### 六大规则总表

| 场景 | 可分动词 (einkaufen) | 不可分动词 (verkaufen) |
|------|---------------------|------------------------|
| 现在时主句 | Ich kaufe ... ein | Ich verkaufe ... |
| 完成时 | Ich habe ... eingekauft | Ich habe ... verkauft |
| zu 不定式 | ... einzukaufen | ... zu verkaufen |
| 情态动词句 | Ich muss ... einkaufen | Ich muss ... verkaufen |
| weil 从句 | ..., weil ich ... einkaufe | ..., weil ich ... verkaufe |
| 规则总结 | 前缀甩句尾/ge 夹中间/zu 夹中间 | 永不分离/不加 ge/zu 放前面 |

### 口诀总结

| 情境 | 口诀 |
|------|------|
| 主句现在时 | 可分前缀甩句尾 |
| 完成时 | 可分 ge 夹中间，不可分不加 ge |
| zu 不定式 | 可分 zu 也夹中间 |
| 情态动词/从句 | 一家人整整齐齐，合体站句尾 |

## 三、Perfekt 完成时

### 公式

```
Perfekt = haben/sein (第二位变位) + Partizip II (句尾)
```

### Partizip II 造法

| 类型 | 规则 | 例 |
|------|------|-----|
| 规则弱变化 | ge + 词干 + t | gemacht, gelernt |
| 强变化 | ge + 变化词干 + en | gegangen, getrunken |
| 可分动词 | 前缀 + ge + 词干 + t/en | eingekauft, aufgestanden |
| 不可分动词 | 前缀 + 词干 + t/en（无 ge）| besucht, verstanden |
| -ieren 结尾 | 词干 + t（无 ge）| studiert, telefoniert |

### 助动词选择

| 用 sein | 用 haben |
|---------|----------|
| 位移动词 (kommen, gehen, fahren, laufen, fliegen) | 及物动词 (machen, kaufen, essen...) |
| 状态变化 (aufstehen, einschlafen, aufwachen, sterben) | 反身动词 (sich freuen...) |
| sein, werden, bleiben | 其余大部分动词 |

## 四、强变化动词 Ablaut 系统

### 核心概念

每个强变化动词需记**三形变 (Stammformen)**：
- Infinitiv → Präteritum → Partizip II

元音按 Ablaut 规律轮换，而非加后缀。

### 七组变化

| 组 | 变化路径 | Partizip II 元音 | 例 |
|------|----------|-----------------|-----|
| 1 | i → a → u | u | trinken → getrunken |
| 2 | ei → ie → ie | ie | schreiben → geschrieben |
| 3 | ie → o → o | o | fliegen → geflogen |
| 4 | e → a → o | o | sprechen → gesprochen |
| 5 | e → a → e | e | essen → gegessen |
| 6 | a → u/a → a | a | fahren → gefahren |
| 7 | o/u/ei → ie → 原元音 | 原元音 | rufen → gerufen |

### 混合动词

元音变了，但词尾是 -t（不是 -en）：denken→gedacht, bringen→gebracht, kennen→gekannt
